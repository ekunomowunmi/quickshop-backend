## Description

Quickshop MVP backend (NestJS + TypeORM + PostgreSQL/PostGIS).

## Database

This project connects to your existing local Postgres:

- Host: `localhost`
- Port: `5432`
- Database: `quickshop_db`
- Username: `postgres`
- Password: `postgres`

Env vars live in `.env`.

### Required for POST /stores (MVP only)

The `stores` table requires `owner_id` (FK to `users.id`). Since there’s no auth yet, this MVP expects you to set an owner user id:

1) Create a user (once) in your DB:

```sql
INSERT INTO users (name, phone, email, password_hash, role)
VALUES ('Dev Owner', '08000000000', 'owner@quickshop.local', 'dev', 'STORE_OWNER')
RETURNING id;
```

2) Put the returned UUID into `.env`:

```bash
QUICKSHOP_OWNER_ID=<uuid-from-query>
```

## Stores endpoints

### Create store

`POST http://localhost:3000/stores`

```json
{
  "name": "Test Store",
  "phone": "08012345678",
  "address": "Lekki",
  "latitude": 6.45,
  "longitude": 3.4
}
```

### Get nearby stores

`GET http://localhost:3000/stores/nearby?lat=6.45&lng=3.4`

Results are sorted by distance using PostGIS KNN (`<->`).
